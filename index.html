What is Browser?

If you are working on HTML, HTM5, CSS, AngularJS or any other client side technology then you can test any of your files as follows:

http://localhost:/index.htm

Above syntax is used to browse index.htm from your home directory at coding ground, similarly you can browse any other file by giving its name in place of index.htm
